Technicalities - What we do

exactly the same thing happening in a browser

Having a list of media for every state of the world, in every state:

  • Run phantojs to the media, monitor all the third party connection performed.
  • Traceroute + GeoIP for every included.
  • the results are send to our infrastrucutre, and exposed via a Rest API
  • historical collection of the results, integration with third party data (disconnect.me)

everyone can help

  • media list: a pull request is enough, (git repo).
  • test results: follow the readme, start the script declaring your country, the results are automatically submitted to our server.

How "users exposition" can be measured ?

  • Any unecrypted connection passing through a foreing owned infrastructure, can be abused without the user has its right (he's not a citizen of the attacking country), this is way we use traceroute.
  • Any data collected by a third party has a certain degree of rights, declaration and duty. Making machine readable those information
